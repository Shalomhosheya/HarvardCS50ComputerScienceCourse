# CS50: Introduction to Computer Science - Harvard University

## Course Overview

CS50 is Harvard University's renowned introductory course to computer science and programming. Taught by Professor David J. Malan, it's designed for beginners with no prior experience, but it's challenging and rewarding. The course emphasizes problem-solving, algorithmic thinking, and good programming practices like correctness, design, and style.

-   **Format**: Self-paced, free online via edX or Harvard's OpenCourseWare.
-   **Duration**: Typically 11 weeks (Week 0 to Week 10), plus a final project. Complete by December 31, 2025, for the 2025 edition.
-   **Languages Covered**: Scratch, C, Python, SQL, HTML/CSS/JavaScript, Flask.
-   **Topics**: Abstraction, algorithms, data structures, encapsulation, resource management, security, software engineering, web development.
-   **Components per Week**: Lecture video, section (discussion), shorts (quick explanations), lab (optional practice), problem set (main assignment).
-   **Assessment**: 9-10 problem sets (graded automatically), final project.
-   **Certificate**: Free honor code certificate; verified certificate available for a fee.
-   **Resources**: All materials at [cs50.harvard.edu/x/2025/](https://cs50.harvard.edu/x/2025/?referrer=grok.com). Join the community on edX, Reddit (r/cs50), or Discord.

The course uses real-world inspired problem sets from arts, humanities, social sciences, and sciences. It's hands-onâ€”expect to code a lot!

## Weekly Breakdown

Each week builds on the previous. Watch the lecture first, then sections/shorts for clarification, do the lab for practice, and tackle the problem set.

### Week 0: Scratch

-   **Main Focus**: Introduction to computational thinking and programming basics using Scratch (a visual drag-and-drop language).
-   **Key Concepts**:
    -   Problem-solving: Inputs, outputs, algorithms.
    -   Basics: Events, loops, conditionals, variables, functions.
    -   Representation: Binary, ASCII, Unicode, RGB colors.
-   **Lecture Highlights**: Fun demo of Scratch projects; overview of computer science.
-   **Problem Set 0**: Build your own interactive Scratch project (game, animation, etc.).
-   **Tips**: Great for beginnersâ€”gets you thinking like a programmer without syntax worries.

### Week 1: C

-   **Main Focus**: Basics of programming in C, a low-level language.
-   **Key Concepts**:
    -   Syntax, compilation, data types (int, float, char, string).
    -   Conditionals (if/else), loops (while, for), functions, libraries.
    -   Command-line tools, debugging with debug50.
-   **Lecture Highlights**: From "Hello, World!" to simple programs; understanding how code runs.
-   **Problem Set 1**: Implement "Hello, World!", Mario's pyramid (loops), and a greedy algorithm for making change (cash/credit card).
-   **Tips**: C is strictâ€”pay attention to semicolons and braces. Use CS50's online IDE (codespace).

### Week 2: Arrays

-   **Main Focus**: Working with collections of data in C.
-   **Key Concepts**:
    -   Arrays, strings as char arrays.
    -   Command-line arguments (argc, argv).
    -   String manipulation, cryptography basics.
-   **Lecture Highlights**: How memory works for arrays; simple ciphers.
-   **Problem Set 2**: Calculate text readability (Flesch index), implement Caesar cipher or substitution cipher.
-   **Tips**: Practice with stringsâ€” they're null-terminated in C (\\0).

### Week 3: Algorithms

-   **Main Focus**: Searching and sorting algorithms.
-   **Key Concepts**:
    -   Searching: Linear search, binary search.
    -   Sorting: Bubble sort, selection sort, merge sort.
    -   Efficiency: Big O notation (O, Î©, Î˜), recursion.
-   **Lecture Highlights**: Visual demos of sorting; why efficiency matters.
-   **Problem Set 3**: Simulate voting systemsâ€”Plurality, Runoff, or Tideman (optional, advanced).
-   **Tips**: Understand time complexity; recursion can be tricky but powerful.

### Week 4: Memory

-   **Main Focus**: Managing memory in C.
-   **Key Concepts**:
    -   Pointers, address-of (&), dereference (\*).
    -   Dynamic allocation (malloc, free), heap vs. stack.
    -   File I/O, hex editing, image formats (BMP).
    -   Tools: Valgrind for memory leaks.
-   **Lecture Highlights**: Memory as a grid; pointers explained with analogies.
-   **Problem Set 4**: Apply image filters (grayscale, sepia, etc.), recover deleted JPEGs from raw data.
-   **Tips**: Pointers are keyâ€”misuse causes seg faults. Practice with small examples.

### Week 5: Data Structures

-   **Main Focus**: Building efficient data storage.
-   **Key Concepts**:
    -   Linked lists, hash tables, trees, tries.
    -   Abstract data types, resizing arrays.
-   **Lecture Highlights**: Why arrays aren't enough; building a dictionary.
-   **Problem Set 5**: Implement a spell-checker (load dictionary, check words) using a data structure.
-   **Tips**: Focus on time/space trade-offs. Hash functions are fun!

### Week 6: Python

-   **Main Focus**: Switching to Python, a high-level language.
-   **Key Concepts**:
    -   Python syntax: Lists, dictionaries, sets.
    -   Libraries (e.g., cs50), exceptions, file I/O.
    -   Translating C code to Python.
-   **Lecture Highlights**: Python's simplicity; same algorithms, less boilerplate.
-   **Problem Set 6**: Rewrite earlier C problems in Python (Mario, cash, readability, etc.), plus DNA sequence analysis.
-   **Tips**: Python is forgivingâ€”great for rapid prototyping.

### Week 7: SQL

-   **Main Focus**: Databases and querying data.
-   **Key Concepts**:
    -   Relational databases, SQL syntax (SELECT, INSERT, UPDATE, DELETE).
    -   Joins, indexes, normalization.
    -   SQLite.
-   **Lecture Highlights**: Tables as spreadsheets; querying large datasets.
-   **Problem Set 7**: Work with movie databases (IMDb-like), solve a mystery with SQL queries.
-   **Tips**: Practice queries interactively. Avoid injection vulnerabilities.

### Week 8: HTML, CSS, JavaScript

-   **Main Focus**: Front-end web development.
-   **Key Concepts**:
    -   HTML structure, CSS styling, JavaScript interactivity.
    -   DOM, events, responsiveness.
-   **Lecture Highlights**: Building simple websites; client-side scripting.
-   **Problem Set 8**: Create a personal homepage with HTML/CSS/JS.
-   **Tips**: Use browser dev tools for debugging.

### Week 9: Flask

-   **Main Focus**: Back-end web development with Python.
-   **Key Concepts**:
    -   Web servers, HTTP, Flask framework.
    -   Templates (Jinja), sessions, forms.
    -   Integrating SQL databases.
-   **Lecture Highlights**: Full-stack apps; MVC pattern.
-   **Problem Set 9**: Build a stock trading web app (Finance).
-   **Tips**: Understand routes and requests.

### Week 10: Cybersecurity

-   **Main Focus**: Ethics, security, and emerging topics.
-   **Key Concepts**:
    -   Common vulnerabilities (SQL injection, XSS).
    -   Privacy, AI ethics, cybersecurity basics.
    -   Emoji as a fun wrap-up (Unicode).
-   **Lecture Highlights**: Real-world hacks; responsible computing.
-   **No Problem Set**: Focus on final project prep.
-   **Tips**: Think about the broader impact of code.

## Final Project

-   **Requirement**: Design and implement your own project using course concepts (e.g., web app, game, tool).
-   **Guidelines**: Original idea, well-documented, video demo.
-   **Submission**: By end of year; graded on design, complexity, style.
-   **Tips**: Start early, scope realistically. Showcase on GitHub.

## Additional Tips for Success

-   **Time Commitment**: 10-20 hours/week.
-   **Tools**: Use CS50's codespace (VS Code in browser), Git for version control.
-   **Help**: Office hours, forums, Stack Overflow (but don't copy code).
-   **Academic Honesty**: Collaborate on ideas, but code must be your own.
-   **Why Take It?**: Builds strong foundations; fun projects; great for careers in tech.
-   **Next Steps**: After CS50, try CS50's Web Programming, AI, or Games courses.
