This repository contains detailed notes covering the entire CS50 course from Harvard University. CS50 is one of the most popular computer science courses in the world, teaching fundamental concepts of computer science and programming.

## üéØ What's Covered

- **Week 0:** Scratch - Computational thinking fundamentals
- **Week 1:** C Programming - Syntax, variables, data types
- **Week 2:** Arrays - Data structures and strings
- **Week 3:** Algorithms - Searching, sorting, Big O notation
- **Week 4:** Memory - Pointers, dynamic allocation, memory management
- **Week 5:** Data Structures - Linked lists, hash tables, trees, tries
- **Week 6:** Python - High-level programming transition
- **Week 7:** SQL - Databases and relational data
- **Week 8:** HTML, CSS, JavaScript - Web development basics
- **Week 9:** Flask - Python web framework
- **Week 10:** Final Project - Putting it all together

**For C Programming:**
```bash

```

**For Python:**

## üéì Course Philosophy

CS50 teaches you to think like a computer scientist:
- **Abstraction:** Breaking complex problems into manageable pieces
- **Algorithms:** Step-by-step solutions to problems
- **Data Structures:** Organizing and storing data efficiently
- **Efficiency:** Writing code that runs fast and uses memory wisely
- **Problem Solving:** Approaching challenges methodically
